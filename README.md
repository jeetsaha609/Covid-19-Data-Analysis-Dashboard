📊 Global COVID-19 Data Analysis Dashboard

🔍 Project Overview

This project presents an interactive data visualization dashboard developed using Tableau to analyze the global impact of COVID-19.

The dashboard provides insights into excess deaths, vaccination trends, country-wise impact, and predictive analysis of future death trends. The objective is to transform raw data into meaningful insights that support data-driven understanding of the pandemic.

---

🎯 Objectives

- Analyze excess deaths to understand the true impact of COVID-19
- Identify the most affected countries globally
- Compare vaccination progress with death trends
- Build predictive insights using forecasting techniques

---

📊 Dashboard Features

1️⃣ Global Excess Deaths Trend

This visualization highlights fluctuations in excess deaths during different phases of the pandemic, helping identify peak impact periods.

2️⃣ Top Affected Countries

Provides a comparative analysis of countries most impacted by COVID-19 based on excess death metrics.

3️⃣ Vaccination vs Death Trend

Shows the relationship between vaccination rollout and mortality trends, indicating how vaccination influenced death rates.

4️⃣ Predictive Death Analysis

Uses Tableau forecasting to estimate future death trends based on historical data patterns.

5️⃣ World Map Visualization

Displays the geographical distribution of COVID-19 impact, enabling quick regional comparison.

---

🛠 Tools & Technologies

- Tableau Public
- Data Visualization
- Forecasting (Tableau Analytics)

---

📈 Key Insights

- Significant spikes in excess deaths during major COVID-19 waves
- Uneven impact across different countries
- Vaccination campaigns contributed to a decline in death rates
- Forecast suggests a gradual stabilization in future trends
